Proyectos de Machine Learning 🤖

Aplicaciones y modelos predictivos usando Python y Scikit-learn

Clasificación de Obesidad

Se entrena un modelo de clasificación para predecir niveles de obesidad a partir de hábitos alimenticios y actividad física. Evaluación con métricas de precisión, recall y matriz de confusión.
Tecnologías: Python, Scikit-learn, Pandas, Matplotlib.

Regresión de Ingresos

Se implementa un modelo de regresión para estimar ingresos a partir de variables como edad, educación y ocupación. Análisis del error cuadrático medio y visualización del ajuste.
Tecnologías: Python, Scikit-learn, Seaborn.

Clasificación con KNN

Se explora el algoritmo KNN para clasificación de flores del dataset Iris. Análisis de hiperparámetros como `k`, y visualización de resultados.
Tecnologías: Python, Scikit-learn, Matplotlib.
