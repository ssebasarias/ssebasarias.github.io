Clusterización de Frijoles Secos: Análisis y Agrupamiento

Este proyecto explora la agrupación de siete variedades de frijoles secos utilizando un sistema de visión por computadora. Se analiza una serie de características físicas como el área, perímetro, redondez, y compactación para identificar patrones y clasificar uniformemente los frijoles según estándares de calidad. El objetivo es facilitar la detección de anomalías y garantizar una clasificación eficiente para el mercado, destacando las propiedades únicas de cada tipo de frijol.
